# neobrutalism-portfolio

Single-page personal portfolio for Akshat Somani — a neobrutalist, pop-art /
comic-book style bento grid built with Next.js (App Router).

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm run start
```

## Deploy

Push to GitHub and import the repo in [Vercel](https://vercel.com/new) —
zero configuration required (no env vars, no API routes).

Design source: see `design_handoff_comic_portfolio/README.md` for the original
handoff notes.
